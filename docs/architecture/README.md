# Mini IVA Studio Architecture Docs

This folder contains High Level Design and Low Level Design documents for the learning MVP platform.

## Documents
- HLD UI: ./HLD_UI.md
- LLD UI: ./LLD_UI.md
- HLD Backend: ./HLD_BACKEND.md
- LLD Backend: ./LLD_BACKEND.md

## Agreed MVP Scope
- Auth: Email OTP, Google sign-in, profile menu, preferences, sign out
- Org and Workspace hierarchy
- Workspace modules:
  - Home
  - Integrations (Proxy Script, Global Variables)
  - Models (create and train)
  - Intents, Alternates, Entities, Triggers CRUD
  - Conversation Flows CRUD + canvas + debug panel
  - Messenger Testing
  - Workspace Settings

## Out of Scope for this MVP
- Version control branches
- Advanced analytics and enterprise governance
